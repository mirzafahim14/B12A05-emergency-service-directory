Emergency Service Directory

## Project Description
This project is a simple emergency service website. Users can see hotline numbers, copy them, and simulate calls using coins.

## Features
- Heart count system
- Copy system
- Call system with coin deduction
- Call history tracking
- Clear history option

## Questions Answer

### 1. getElementById vs querySelector
getElementById is used to select a single element by ID. querySelector can select any element using CSS selector.

### 2. Create element
We use document.createElement() and appendChild().

### 3. Event Bubbling
Event moves from child to parent.
### 4. Event Delegation
Using parent event listener to handle child elements.

### 5. preventDefault vs stopPropagation
preventDefault stops default behavior, stopPropagation stops event bubbling.
